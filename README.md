## react-init-cli

##### 快速生成 react+typescript 项目的脚手架工具。

### 安装

```
npm install -g react-init-cli
```

### 使用

```
react-init-cli init my-react
```

### 模版类型

1. `init`为通用项目模版，配置了常用的 react 工具。

##### 初始化项目完成以后

```
cd my-react //进入项目目录
yarn install //安装项目依赖
npm start //运行项目
```

### 命令

###### 启动项目

`npm start`

###### 打包项目

`npm run build`
